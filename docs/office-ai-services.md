## 3. 업무 자동화 상세

### 3.1 견적서 자동 생성
- **시간 절약**: 30분 → 3분 (90% 단축)
- **프로세스**: 고객명·품목·수량 입력 → AI 단가 계산 → PDF/Excel 자동 출력
- **특징**: 회사 양식 맞춤, 할인율·납기일 자연어 수정 가능

### 3.2 세금계산서 자동 발행
- **시간 절약**: 20분 → 2분 (90% 단축)
- **프로세스**: 거래 내역 업로드 → AI 세율 적용 → 국세청 양식 생성
- **특징**: 법적 요건 만족, 정확한 세액 계산

### 3.3 정부지원사업 사업계획서
- **시간 절약**: 2주 → 1시간 (95% 단축)
- **프로세스**: 사업 개요 입력 → AI 시장분석·기술개발계획 작성 → PPT 생성
- **특징**: 정부 양식 준수, 발표자료 자동 포함

### 3.4 계약서 자동 작성
- **시간 절약**: 1시간 → 5분 (92% 단축)
- **프로세스**: 계약 조건 입력 → AI 법적 조항 검토 → 표준 양식 완성
- **특징**: 법적 리스크 검토 포함

### 3.5 월말 회계 정산
- **시간 절약**: 4시간 → 30분 (87% 단축)
- **프로세스**: 거래 내역 업로드 → 계정과목 자동 분류 → 재무제표 생성
- **특징**: 손익계산서, 재무상태표 자동 완성

### 3.6 KS 인증 서류 작성
- **시간 절약**: 1개월 → 3일 (90% 단축)
- **프로세스**: 회사 정보 입력 → KS 규격 매뉴얼 작성 → 심사 Q&A 생성
- **특징**: KS 심사위원 출신이 설계, 통과율 95% 이상

## 7. LaunchKit (AI 사업계획서 자동 생성)

- AI 기반 사업계획서(IR) 및 랜딩페이지 자동 생성 SaaS
- 인터뷰 형식으로 사업 정보 입력 → AI가 전문 IR 문서 생성
- 요금: Free(100토큰, 0원) / Pro(500토큰, 3만원) / Pro Max(2,500토큰, 5만원)
- 작업별 소비 토큰: 간단 IR=80, 상세 IR=200, 간단 랜딩=120, 상세 랜딩=300
