03  |  게임 추천 시스템
🎯 추천 흐름
1️⃣
인원수 선택2명 ~ 10명+ 버튼 탭
2️⃣
연령대 필터아동(~10) / 청소년(11~17) / 성인(18+) / 전체
3️⃣
카테고리 선택전략·카드·파티·협력·추리·경제·공포·롤플레이
4️⃣
결과 카드 표시이미지·난이도·플레이시간·재미도 ★★★★☆
5️⃣
상세 보기규칙·팁·YouTube 영상·AI 검색 자동 입력
🗄️ DB 테이블 구조
📦 board_games
name / name_en
min_players / max_players
play_time_min / difficulty / fun_score
categories[]   age_min
image_url / description
rules / tips / youtube_url
📦 hand_games
name / min_players / max_players
categories[]   age_min
description / rules / tips
materials (도구 없이 하는 게임)
04  |  게임 DB 관리  (GBHQ)
🎲
보드게임 관리
· 게임 목록 카드형 UI
· 수정 아이콘(우상단) + 읽기전용 상세보기
· AI 검색 → 이름 입력 시 모든 필드 자동 채우기
· Wikipedia 이미지 자동 가져오기
· YouTube 인라인 재생 지원
✋
맨손 게임
· 도구 없이 사람끼리 하는 게임
· 별도 테이블 (hand_games)
· 카테고리: 언어·신체·추리·창의·조용·시끌벅적
· 인원수·연령대 필터 동일 적용
🧑‍🤝‍🧑
방문자 & 방문기록
· QR 촬영으로 프로필 사진 업로드
· 이모지 아바타 선택 (피커 팝업)
· 방문기록 카드형 UI (3개월~전체 필터)
· 단체사진 업로드 최대 5장, 개별 삭제
· 방문기록 ↔ 게임 호출 이력 연동
📋
호출 메뉴 관리
· 메뉴 아이템 등록 (이름 + 이모지 피커)
· 솔드아웃(품절) 표시 — 빨간 SOLD OUT 뱃지
· 아이들 → 텔레그램으로 집주인 호출
· 집주인 → 아이들 메시지 모달 수신
· 확인 시 텔레그램 자동 답장 + 알림음 반복
05  |  방문자 관리
📲 체크인 흐름
1.
이름 입력기존 방문자 자동완성 지원
2.
나이 입력연령대 필터에 자동 반영됨
3.
사진 방식 선택① QR 촬영  ② 이모지 아바타 선택
4.
QR 촬영일회용 토큰 생성 → 폰 카메라 → Supabase Storage
5.
체크인 완료방문기록 자동 생성 + 환영 팝업

사진 삭제단체사진 개별 삭제 버튼(X)

visitors: id · name · age · avatar · memo

07  |  음악 플레이어  (Spotify 연동)
🎵 핵심 기술: iframe SPA 방식
일반 SPA에서 페이지를 이동하면 Spotify 재생이 끊깁니다.
이를 해결하기 위해 메인 페이지를 Spotify 플레이어 호스트로 고정하고, 나머지 콘텐츠를 <iframe> 내에서 로드하는 방식을 적용했습니다.
🔍
통합 검색곡·앨범·플레이리스트 전체 검색
▶️
자동 연동Spotify Embed API autoplay=1
📱
반응형 배너빔프로젝터 모드 확대 지원
🎶
로그인 여부비회원 미리듣기 / 회원 전곡 재생
🔄 페이지 이동 흐름
▶
메인 페이지 로드
Spotify iframe 초기화 + 재생 시작
↓
📄
콘텐츠 영역
별도 <iframe>으로 서브 페이지 렌더링
↓
🔗
링크 클릭
location.href 인터셉트 → iframe.src 변경
↓

08  |  집주인 호출  &  텔레그램 연동
📱 아이들 → 집주인
①
호출 메뉴에서 아이템 탭 (예: 🍕 피자 주세요)
②
앱에서 Telegram Bot API 직접 호출
③
집주인 텔레그램으로 주문 내용 도착
④
방문기록에 호출 이력 자동 저장
💬 집주인 → 아이들
①
집주인이 텔레그램에서 메시지 전송
②
앱이 n8n Webhook으로 메시지 수신
③
모달 팝업 + 알림음 반복 재생
④
확인 버튼 → 텔레그램 자동 답장
🛒 호출 메뉴 특별 기능
솔드아웃품절 처리 → 빨간 SOLD OUT 뱃지 + 탭 비활성화
이모지 피커메뉴 등록 시 이모지 선택 팝업
알림음 반복집주인 메시지 수신 시 확인 전까지 반복
호출 히스토리item_label · item_emoji · note 저장
📡 기술 흐름
아이들 호출 → Telegram Bot API 직접 전송
집주인 메시지 → n8n Webhook → 앱 폴링 수신
Bot Token: config.js 에 저장 (프론트 직접 호출)
